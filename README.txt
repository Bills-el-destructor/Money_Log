This is a small system made with Laravel about movement registration in a small business. 
It contains a navigation bar, as well as a separate user authentication. 
In the frontend you are using Bootstrap 3.4.1 using your cdn.
The system can list the movements in addition to creating and modifying them. 
This also applies to upload or edit an image.

-----------------------------------------------------------------------------------------

Este es un pequeño sistema hecho con Laravel sobre el registro de movimientos en una pequeña empresa. 
Contiene una barra de navegación, así como una autenticación de usuario separada. 
En la interfaz está usando Bootstrap 3.4.1 usando su cdn.
El sistema puede enumerar los movimientos además de crearlos y modificarlos. 
Esto también se aplica a cargar o editar una imagen.
